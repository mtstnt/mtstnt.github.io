Design System Lab is a practical sandbox for building and validating reusable UI primitives.

## Scope

- Semantic color token mapping for light and dark modes
- Reusable composition patterns for cards, navigation, and content sections
- Documentation pages authored in Markdown and MDX

## Outcome

The project reduced design drift and made new page creation faster with clear defaults.
